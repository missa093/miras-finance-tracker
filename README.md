# Личный финансовый трекер

Статический сайт для Netlify. Откройте `index.html` локально или загрузите репозиторий на Netlify.

## Где хранятся данные

Данные сохраняются в `localStorage` конкретного браузера. Это удобно для личного трекера без сервера: сайт работает быстро, бесплатно и без базы данных.

Важно: данные не синхронизируются между устройствами автоматически. Для переноса на другой телефон или браузер используйте кнопки `Экспорт` и `Импорт`.

## Деплой на Netlify

1. Загрузите проект в GitHub.
2. Создайте новый сайт в Netlify из этого репозитория.
3. Build command оставьте пустым.
4. Publish directory: `.`
